# CHED RAISE 2025

Official website for the **CHED RAISE 2025: Responding through AI for Societal Empowerment** conference.

## Overview
This project is a modern, responsive website built to promote the CHED RAISE 2025 conference. It features a dark, glassmorphic design and provides comprehensive information about the event's vision, objectives, program, and partners.

## Tech Stack
- **Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS (v3)
- **UI Architecture:** Shadcn UI
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Features
- **Modern Aesthetic:** Dark mode with glass effects, glowing gradients, and grid patterns.
- **Scroll Animations:** Smooth entry animations for all sections.
- **Interactive Elements:** Tabbed program view, hover effects on cards, and mobile-responsive navigation.
- **Content Sections:**
  - **Hero:** Event details and calls to action.
  - **About:** Rationale and vision (NAIS-PH & ACHIEVE Agenda).
  - **Objectives:** RAISE framework pillars.
  - **Program:** Multi-track agenda for Students, Teachers, and Admins.
  - **Partners:** Strategic partners and lead institutions.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd ched-raise
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```

### Build
Build for production:
```bash
npm run build
```

## Project Structure
- `src/components`: React components for each section (Hero, About, etc.).
- `src/components/ui`: Shadcn UI primitives.
- `src/lib`: Utility functions.
- `public`: Static assets (images, SVGs).
